## Shortcuts

GLYPH        | NAME
-------------|----------------------------------------
COMMAND      | Command, Cmd, Clover, (formerly) Apple
ALT          | Option, Opt, (Windows) Alt
SHIFT        | Shift
CTRL         | Control, Ctl, Ctrl

COMMAND (CTRL) - ,               | preferences

COMMAND (CTRL) - P               | print
